# SWITCHER

场景切换示例，基于 [swiper](swiper-repo) 和 [anime.js](anime-repo)，可基于该项目初始化实现滑动场景切换的项目。

## 开始

```bash
# 安装依赖
npm install # 或者使用 yarn install

# 开发模式
npm start

# 构建
npm run build
```

## 开发

开发时，主要涉及到三个文件

- `src/index.html`

    整个网页的 HTML 文件

- `src/js/main.js`

    Javascript 文件

- `src/css/main.scss`

    样式文件

[swiper-repo]: https://github.com/nolimits4web/Swiper
[anime-repo]: https://github.com/juliangarnier/anime
